zTree v2.6 Node VS v3.0 Node

  • 1. Overview

    • zTree v3.0 - zTree v2.6: No major changes to the node structure. Mainly to the 'nodes' changed to 'children', also removed the 'parentNode' attributes, to avoid infinite loop caused by clone data.
