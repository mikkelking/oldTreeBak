zTree v2.6 Setting VS v3.0 Setting

  • 1. Overview

    • zTree v2.6: setting is very messy, mixed with attributes of various types, is not conducive to search, is not conducive to upgrade.
      zTree v3.0: The structure of the setting has been adjusted. As far as possible to see the property name to know its functionality, added some configuration attributes, and deleted the attributes which can't be set. (treeId / treeObj excluded)

  • 2. Detailed

    • It is arranged order by the zTree v3.0 structures. Because of limited space and time, please view the API documentation for specific instructions.

      setting.treeId (v3.0) === setting.treeObjId (v2.6)
