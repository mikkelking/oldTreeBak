How to Solve Css Style Error?


  •     There are many factors resulting to style conflict, generally summed up as follows:
  • 1. Cause of Issue:
    • Css conflict (in page and in zTree)
    • Css conflict (in other plug-in and in zTree)
    • Modify zTree's css has some error.
  • 2. Solution:
        Because there are too many conflicting factors, the following tips are recommended to reduce the conflict:
    • A. Avoid using 'id' set the style, try using 'class'. Because the use of 'id' set style level is too high, zTree style will be overwritten.
    • B. zTree default css style definitions will not have to re-set all, so inevitably there are omissions. If there is missing the style affected the results, then to increase in zTreeStyle setting. And if this happens, I hope you can inform me, thank you.
    • C. See if there are naming conflict of, leading to css conflict.
    • D. Finally, to see whether your modified style has any error.
    • E. Above answers are invalid, then please contact me.
