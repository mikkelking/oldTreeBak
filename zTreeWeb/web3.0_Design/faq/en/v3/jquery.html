jQuery Version Requirement?


  •     Basically there is no limit. When develop zTree, I use jQuery 1.4.4. Now, tested jQuery 1.3+ should be able to work properly, if you find any compatibility problems, please notify me. Email: hunter.z@263.net

  •     If you want to have better animation, better operating efficiency, you can choose jQuery 1.8.x or later; If you want to have to reduce the js code, use the basic features, you can select the mini version of an earlier version of jQuery. So please choose your own needs.

  •     What zTree provides is uncompressed jQuery 1.4.4. The goal is help to debug.
        * To download jQuery, please visit: http://docs.jquery.com/Downloading_jQuery
