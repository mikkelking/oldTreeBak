JSON String / Object


  •     Some friends do not understand the difference between JSON String and Object, so make an error when using zTree. Here is the explaination:

  • (1) JSON Object is an js object which has JSON data format
        For example: {name:"abc"}
        A standard JSON data is that its attribute names must be use single or double quotes wrapped.
        For example: {"name":"abc"}

  • (2) JSON-formatted string -- It is just a string, just JSON-formatted string
        For example: '{"name":"abc"}'

  •     See the differences between two examples? Obviously, the key problem lies in the quotes.

  •     So please do not increase the two quotes outside the node data!
