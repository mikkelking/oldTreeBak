技术支持

zTree 技术支持信息

  • 如果您有好的建议和想法，也非常欢迎随时与我们联系。
    • 在使用过程中有任何问题，可通过以下相应的方式与我们联系，技术支持是免费的。

    注意事项：
      1、请先在 zTree 的 API 文档中查找一下相应的帮助说明。
          (请在下载的 zTree zip 文件中获取 API 文档)
      2、给我们留言时，请将问题描述清楚，以便尽快发现问题原因。
      3、发 Email 时，最好能把出错的 HTML 例子附上。
  • 常见问题
    有疑问？先看看常见问题里面有没有您遇到的情况吧。
  • 博客
    有什么问题可以直接到博客来留言，我们会及时进行答复的。
  • 发现Bug了？
    想要提Bug，可以来 Google 的项目管理页面提交，当然了也可以直接去上面的博客告诉我们。
  • Email联系
    如果您没有百度账号，也没有Google的账号，那么最简单的方法是给我们发Email，我们会及时进行答复的。
