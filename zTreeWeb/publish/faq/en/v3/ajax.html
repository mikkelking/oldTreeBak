    • Be accurate to say: "What is AJAX?" -- Ajax (an acronym for Asynchronous JavaScript and XML) is a group of interrelated web development methods used on the client-side to create asynchronous web applications. With Ajax, web applications can send data to, and retrieve data from, a server asynchronously (in the background) without interfering with the display and behavior of the existing page.
