GameTree API Document

source file: gameTree.js

dependent lib: jQuery v1.4.4+, zTree v3.5

demo file: gameTree.html

GameTree.addLeftRoot()

add root in the left tree

1) params

none

2) return: None

GameTree.initLeft(nodes, options)

init the left tree

1) params

nodes (Array)
The nodes data of the left tree.
Please set the 'isParent=true' to folder's data.

  1. example:
  3. [
  4. { id: 1, pId: 0, name: "Folder 001", externalID: 123, isParent: true, open: true, power: true},
  5. { id: 11, pId: 1, name: "Folder 011", externalID: 123, isParent: true, power: true},
  6. { id: 1111, pId: 111, name: "Game 011", externalID: 95501, externalTime: 'yesterday', power: true}
  7. ]

options (JSON)
Setting of the left tree
Mainly to set the callback.

  1. example:
  3. {
  4.     treeId: 'leftTree',  //The left tree's id
  5.     callback: {
  6.         //When user change folder or game's status, will trigger it
  7.         changeStatus: function(nodes, status) {},
  9.         //When add folder or game, will trigger it
  10.         //the callback can update nodes's id, please return like:
  11.         // {tId:xxx, id:yyy}
  12.         add: function(nodes, callback) {},
  14.         //When remove a folder or game, will trigger it
  15.         remove: function(nodeId) {},
  17.         //When rename a folder or game's name, will trigger it
  18.         rename: function(node) {},
  20.         //When drag&drop to change a folder or game's position, will trigger it
  21.         sorted: function(nodes) {}
  22.     }
  23. }

2) return: None

GameTree.initRight(nodes, options)

init the right tree

1) params

nodes (Array)
The nodes data of the right tree.
Please set the 'isParent=true' to folder's data.

  1. example:
  3. [
  4. { id: 1, pId: 0, name: "Folder 001", externalID: 123, isParent: true, open: true, status: 1},
  5. { id: 1111, pId: 1, name: "Game 011", externalID: 96501, externalTime: 'yesterday 12:13', status: 2},
  6. { id: 1112, pId: 1, name: "Game 012", externalID: 96502, externalTime: '2014-11-11 11:11', status: 3}
  7. ]

options (JSON)
Setting of the right tree
Mainly to set the callback.

  1. example:
  3. {
  4.     treeId: 'rightTree',  //The right tree's id
  5.     callback: {
  6.         //When user change folder or game's status, will trigger it
  7.         changeStatus: function(nodes, status) {}
  8.     }
  9. }

2) return: None

GameTree.expandLeft(flag)

expand/collapse all folder of the left tree

1) params

flag (Boolean)
flag = true will expand all folder.
flag = false will collapse all folder.

2) return: None

GameTree.expandRight(flag)

expand/collapse all folder of the right tree

1) params

flag (Boolean)
flag = true will expand all folder.
flag = false will collapse all folder.

2) return: None
